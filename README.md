# Trading Bot Dotnet

Web application that provides real-time cryptocurrency price quotes with continuously updated data.

## Overview

This project is a website that provides real-time cryptocurrency prices using SignalR, a real-time communication technology from .NET.

## Technologies

- .NET 9.0
- React
- Extern API (Binance API)
- Real Time Communication (SignalR)

## Backend Installation

Clone the repository:

```bash
git clone https://github.com/NaelNunes/trading-bot-dotnet.git
cd trading-bot-dotnet
```

Restore Dependencies:

```bash
dotnet restore
```

Build the backend:

```bash
dotnet build
```

Run the backend:

```bash
dotnet run
```

## Frontend Installation

Navigate to the frontend folder:

```bash
cd frontend
```

Install the dependencies:

```bash
npm install
```

Running the frontend:
```bash
npm start
```

The React application will be available at:
```bash
http://localhost:3000
```
